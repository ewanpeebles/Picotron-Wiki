# The Picotron Wiki

> [!IMPORTANT]
> This is a community project. We are not affiliated with [Lexaloffle Games](https://www.lexaloffle.com/).

![Picotron Wiki Label Image](assets/picotron_wiki_label.png)

This repository contains data for the [Picotron](https://www.lexaloffle.com/picotron.php) Wiki cartridge.

You can integrate our Wiki Viewer cartridge directly into the Picotron desktop by downloading the it [here](https://www.lexaloffle.com/bbs/?tid=140925), or running the following commands in the terminal:

```
load #picotron_wiki
cd desktop
save picotron_wiki-11
```

Notice something that we missed? Open a [new issue](https://github.com/Picotron-Software-Developers/Picotron-Wiki/issues) describing what we could change.
Or, if want, you can [fork our repository](https://github.com/Picotron-Software-Developers/Picotron-Wiki/fork), add your changes, and then open a pull request.
